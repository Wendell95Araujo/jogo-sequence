Sobre Este Jogo

Bem-vindo ao Sequence Multiplayer! Este projeto foi desenvolvido como uma forma de praticar e aprimorar habilidades de desenvolvimento web, incluindo JavaScript, HTML, CSS e a integração com tecnologias de tempo real como o Firebase.

O objetivo é oferecer uma versão digital, gratuita e de fácil acesso do clássico jogo de tabuleiro Sequence, permitindo que amigos e familiares joguem juntos, não importa onde estejam. O jogo foi projetado para ser intuitivo e divertido, com a opção de jogar contra bots ou desafiar outros jogadores online.

Este é um projeto em constante desenvolvimento. Agradecemos seu feedback e esperamos que você se divirta!
